== Description ==

Ipush plugin helps you integrate with [InstantPush](https://instantpu.sh) services on your wordpress website.

= Company =
Instantpush is a web push notification management service for publishers and website owners. You can send unlimited push notifications to unlimited subscribers for free.

= Features =
* Create subscriber segments, automate push notification delivery and track impressions and clicks.
* The platform is designed for engaging and re-targeting end users in a efficient way to achieve desired goals.
* The platform also provides a way to monetise your website traffic by delivering third party advertisements to your subscribers.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ipush` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the InstantPush screen to configure the plugin.

== Frequently Asked Questions ==

= How to use this plugin to send notifcations? =

1. Register at [InstantPush](https://instantpu.sh/user/register).
2. Create your website's Adspace.
3. Create user segment/group to send notifications to.
4. Create notification structure that you would like to show to website's users.
5. Add created Adspace ID, Segment ID and Notification ID in InstantPush settings menu in admin panel.
6. Tick the Send InstantPush Notification box shown in post editor.
